## 数据流

### 对话整理流程（MemoryAgent）★已实现

```
Java 端检测到对话 ≥ 阈值
    │
    ▼
POST /ai/memory/consolidate
    │
    ▼
MemoryAgent.run()
    │
    ├── 构建消息（含旧偏好/未完成 + 新N条对话）
    │
    ├── LLM function calling 循环
    │   ├── LLM 读取对话，提取候选事实关键词
    │   ├── LLM 主动调用 search_similar_facts 工具
    │   │   └── FactRetrievalTool._execute(queries)
    │   │       └── VectorService.search_by_text() (并发)
    │   │           └── TextEmbedding.embed() → Redis 向量检索
    │   ├── 工具结果返回 LLM
    │   └── LLM 判断冲突 → 输出结构化 JSON
    │
    ├── _extract_json() 解析LLM输出（兼容markdown代码块）
    │
    ├── _store_facts_to_vector() → 新事实向量化入库
    │   └── TextEmbedding.embed() → VectorService.add_vector()
    │
    └── 返回 MemoryConsolidateResponse（new_facts + superseded_ids + ...）
```

### 聊天对话流程（CHAT模式）★已实现

```
用户输入
    │
    ▼
POST /ai/chat 或 /ai/chat/stream
    │
    ▼
OrchestratorAgent.run() 或 run_stream()
    │
    ├── _resolve_mode(): 从context提取用户指定的mode
    │
    ├── mode=CHAT → IntentionRecognizer.recognize()
    │   │   ├── recognizer.py → LLM识别
    │   │   └── LLM失败时 → fallback.py 关键词兜底
    │   └── map_intention_to_mode() 映射为 AgentMode
    │
    ├── mode!=CHAT → 直接使用用户指定的mode
    │
    └── _dispatch(mode)
        ├── CHAT → _execute_chat() ★已实现
        │   └── BaseAgent._build_messages() → _call_llm() → _process_response()
        │
        ├── RETRIEVAL → TODO: 等待 RetrievalAgent 实现
        ├── DIAGNOSIS → TODO: 等待 DiagnosisAgent 实现
        ├── GUIDANCE → TODO: 等待 GuidanceAgent 实现
        └── FULL → TODO: 等待 chains/pipeline.py 实现
```

### 知识检索流程（独立调用）★已实现

```
KnowledgeRetrievalTool.run(query, top_k)
    │
    ├── 1. TextEmbedding.embed(query)
    │   └── 1024维向量
    │
    ├── 2. VectorService.search(vector, top_k)
    │   └── Redis ANN → TopK 文档
    │
    └── 3. 格式化为 List[VectorSearchResult]
        └── 返回 {id, score, content, metadata}
```

## 异常处理架构

```
BaseTool.run()  ←───────── 模板方法，统一异常处理
    │
    ├── try: _execute() → ToolResult(success=True, data=...)
    ├── except ToolException: → ToolResult(success=False, error=ToolError(code, message))
    └── except Exception: → ToolResult(success=False, error=ToolError(code="TOOL_ERROR"))

BaseAgent.run()  ←───────── 模板方法，统一异常处理
    │
    ├── try: _build_messages() → _call_llm() → _process_response()
    └── except Exception: → AgentOutput(metadata={error_type, error_detail})

MemoryAgent.run()  ←─────── 覆盖父类，自定义异常处理
    │
    ├── try: chat_with_tools() → except → AgentOutput(status="error")
    ├── try: _extract_json() → except → 降级为brief_summary
    └── try: _store_facts_to_vector() → except → 静默跳过（不影响主流程）

OrchestratorAgent.run()  ←── 覆盖父类，自定义异常处理
    │
    └── try: 意图识别 → 路由分发 → except → AgentOutput(status="error")
```
